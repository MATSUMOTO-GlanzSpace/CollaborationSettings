# CollaborationSettings

コラボレート開発（共同開発）のための資料集

## 概要

このリポジトリは公開リポジトリですが、リポジトリ作成者のみが編集できる設定になっています。

## リポジトリの設定

- **公開設定**: Public（一般公開）
- **編集権限**: リポジトリ作成者のみ
- **閲覧権限**: 誰でも可能

## リポジトリの目的

共同開発のための資料やガイドラインを一般に公開しながら、内容の品質管理のため編集権限をリポジトリ作成者に限定しています。

## 設定方法

GitHubリポジトリをこの設定にする方法については、[SETTINGS.md](SETTINGS.md)を参照してください。

## コントリビューション

このリポジトリは閲覧専用です。提案や改善案がある場合は、Issueを作成してください。
